# Learning Notes

This directory stores learning material for AI Agent engineering, project retrospectives, and interview preparation.

Use it to connect Fitness Agent development work with concepts such as model + harness, tool use, MCP, observation, memory, context management, action interfaces, permissions, and safety boundaries.

Recommended structure:

- `prompt-library.zh.md`: reusable prompts for development, review, learning, and interview practice.
- `stage-retrospectives/`: stage-level engineering retrospectives for non-trivial feature or architecture work.

Do not store real health data, API keys, database files, private paths, or other sensitive information here.
